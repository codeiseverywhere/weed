Phylactery by HUEMOR

A CSS framework for building custom designed websites.

User Interface Widgets

  • Typography
  • Buttons
  • Form elements styling (input, select, etc)
  • Page borders
  • Breadcrumbs
  • Site Headers (navigation bars)
  • Site Footers
  • Video player (HTML5, YouTube, and more!)
  • Modal pop-ups
  • Tabs
  • Collapses

Interactive Effects

  • Scroll-affixed columns
  • Scroll effects & animations
  • HTML-based infinite scroll
  • Page transitions
  • Google Maps

  • Historical

  • WOW.js based scroll animations

Layout & Reusable Layout Patterns

  • Grid
  • Hero Headers
  • Generic content section templates
  • Triptychs (multi-column key info blocks)
  • Staff/Employee Grids
  • Obi strip (callout footers)

Component & Layout Examples

  • Marketing Page (triptych & multiple content sections)
  • About/Company Page
  • Blog Article layout
  • Contact Page
